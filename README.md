# ⚡ Detecção de Perdas e Fraudes

> Detecção de perdas não técnicas e fraudes (gatos, adulteração de medidor) com IA e análise de consumo, recuperando receita e reduzindo perdas na distribuição.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/energia-deteccao-perdas/

## 📊 Resultados-chave (KPIs)
**93%** Detecção · **-28%** Perdas · **R$ 24M** Receita rec. · **-40%** Inspeções

## 🧠 Tecnologias de IA
ML · Anomaly Detection · Smart Meter

## ✨ O que faz
- Detecção de fraude de consumo
- Análise de smart meter
- Priorização de inspeção
- Recuperação de receita
- Redução de perdas

## 🗄️ Fonte de dados real
[ANEEL + smart meters](https://www.gov.br/aneel) — bases de entrada e resultados disponíveis para download em CSV/Excel no app.
